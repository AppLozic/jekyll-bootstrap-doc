Android

Open source Android Chat SDK that lets you easily add chat into your application using a gradle dependency.

iOS

Open source iOS Chat framework lets you easily add chat into your application, supports both Objective-C and Swift.

Web

Javascript Chat Plugin lets you easily add chat into your website. Fully responsive UI, supports mobile website and notifications.

PhoneGap

Open source PhoneGap Chat Plugin lets you add chat into apps built using PhoneGap, supports Push Notifications.

Platform APIs

Platform API's that enables your server to send messages to your users or integrate any external services.

Configuration

Configuration details for setting up webhook, push notification and other functionalities in Applozic Dashboard.
