Android Chat SDK

Applozic Android Chat SDK lets you easily add chat into your application using a gradle dependency. You can build chat and messenger apps like whatsapp in no time. You can customize theme through androidmanifest file, change colors, chat bubble, etc through UI toolkit. You are free to fork our Github repository and build anything on top of it.

iOS Chat SDK

Applozic iOS Chat framework lets you easily add chat into your application. It comes with customizable UI components. Download the framework and add to your project.
Get Started

Javascript (Web) Chat Plugin

Applozic Javascript Chat Plugin lets you easily add chat into your website. It comes with a full responsive UI. You can also design your own UI using a light weight version available in Github.
Get Started

PhoneGap Chat Plugin

Applozic PhoneGap Chat Plugin lets you add chat into apps built using PhoneGap. It comes with a full responsive UI and supports PhoneGap Push Notifications. You can also design your own UI using a light weight version available in Github.
Get Started
